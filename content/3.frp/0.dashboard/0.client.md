## 开启客户端管理 UI

您可以通过如下增加配置到配置文件开启客户端管理 UI:

:::code-group

```json [JSON]
{
  "webServer": {
    // 监听地址和端口
    "addr": "127.0.0.1",
    "port": 7500,
    // 可选，BasicAuth 认证配置
    // 开放公网访问请务必启用，避免未经授权的修改
    //"user": "<USER>",
    //"password": "<PASSWORD>"
  },
}
```

```ini [TOML]
# 监听地址和端口
webServer.addr = "127.0.0.1"
webServer.port = 7500
# 可选，BasicAuth 认证配置
# 开放公网访问请务必启用，避免未经授权的修改
#webServer.user = <USER>
#webServer.password = <PASSWORD>
```

```yaml [YAML]
webServer:
  # 监听地址和端口
  addr: "127.0.0.1"
  port: 7500
  # 可选，BasicAuth 认证配置
  # 开放公网访问请务必启用，避免未经授权的修改
  #user: "<USER>"
  #password: "<PASSWORD>"
```

:::

更多配置项请参考 Frp 官方配置说明: [https://gofrp.org/zh-cn/docs/features/common/ui/](https://gofrp.org/zh-cn/docs/features/common/ui/)

## 概览

概览界面列出了当前活动的隧道，以及相关状态。

点击 `刷新` 可刷新当前页面数据。

## 客户端配置

在客户端配置界面，您可以查看当前配置，并动态更新配置。

点击 `刷新` 可刷新当前页面数据。  
修改配置后，点击 `保存并热重载` 可动态更新当前配置。
