## 命令行参数说明

我们的修改版本 Frp 客户端，新增了以下命令行参数：

| 参数   | 说明                                   |
|------|--------------------------------------|
| `-u` | 简易启动功能参数。用户的访问令牌。                    |
| `-p` | 简易启动功能参数。隧道 ID 列表，多个 ID 之间用半角符号逗号分隔。 |

其他参数与官方版本 Frp 客户端参数相同。
