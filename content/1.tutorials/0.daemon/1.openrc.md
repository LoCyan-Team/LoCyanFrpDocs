title: OpenRC
description: 使用 OpenRC 管理和守护进程。
---
我们假设你的 Frp 客户端和配置文件在 `/opt/lcf/` 中，配置文件名为 `frpc.ini`

## 给与 Frp 客户端运行权限

```shell
chmod +x /opt/lcf/frpc
```

## 写入系统服务

在 `/etc/init.d/` 中新建一个文件，并写入以下内容

```shell [lcf-frpc]
#!/sbin/openrc-run

description="LoCyanFrp Client"
command="/opt/lcf/frpc"
command_args="-c /opt/lcf/frpc.ini"

pidfile="/run/${RC_SVCNAME}.pid"
```

## 赋予脚本执行权限

```shell
chmod +x /etc/init.d/lcf-frpc
```

## 启动并设置自动启动

```shell
service lcf-frpc start
rc-update add lcf-frpc
```
