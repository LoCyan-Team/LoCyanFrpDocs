## 视图

### 卡片视图

隧道卡片展示了隧道的基本信息，如隧道名、隧道 ID、隧道协议、隧道所在节点服务器等。

## 查看隧道信息

点击 `详细信息` 即可查看隧道的详细信息。

### 详细信息说明

| 数据             | 说明                   |
|----------------|----------------------|
| 服务器 ID         | 隧道的 ID。              |
| 隧道名称           | 隧道的名称。               |
| 连接地址           | 隧道的连接地址。             |
| 隧道类型           | 隧道穿透协议类型。            |
| 节点名称           | 隧道所在节点服务器名称。         |
| 服务器 IP         | 隧道所在节点服务器 IP。        |
| 节点主机名          | 隧道所在节点服务器主机名。        |
| 节点 IP          | 隧道所在节点的 IP 地址。       |
| 本地 IP          | 隧道本地服务 IP。           |
| 本地端口           | 隧道本地服务端口。            |
| 远程端口           | 隧道远程服务端口。            |
| 自定义域名          | 隧道 vHost 绑定域名。       |
| 路径映射           | 隧道自定义 vHost 访问路径。    |
| Proxy Protocol | Proxy Protocol 协议版本。 |

## 编辑

点击 `修改` 按钮即可修改隧道信息。

修改信息的参数，请参考 [创建隧道](./1.create.md)。

## 删除隧道

点击 `删除` 即可删除当前隧道。

## 强制下线隧道

点击 `强制下线` 即可强制下线隧道。

::note
隧道强制下线后，由于 Frp 客户端设计，如您使用的不是我们的定制客户端，将会自动重连。原版为避免重连导致的问题，若被下线后，隧道将被短暂加入禁止连接列表。
::
